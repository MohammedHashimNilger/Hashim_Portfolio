# Mohammad Hashim — Portfolio Website

A full-featured portfolio built with React + Vite + TypeScript, inspired by and structurally identical to the reference site.

## Tech Stack
- React 18 + TypeScript + Vite
- GSAP + Lenis (smooth scroll + animations)
- React Router DOM
- React Icons, React Fast Marquee

## Setup

```bash
npm install
npm run dev
```

## Customization

All personal data is in **`src/config.ts`** — edit your name, bio, projects, experience, social links there.

## Images needed in `/public/images/`
- `mypicnbg.png` — your photo (no background, for mobile view)
- `resumecraft.png` — ResumeCraft project screenshot
- `opticvista.png` — OpticVista screenshot
- `examseating.png` — Exam Seating System screenshot
- `codereviewer.png` — AI Code Review Agent screenshot
- `ngo.png` — NGO Membership System screenshot

## Video needed in `/public/video/`
- `video.webm` — background video for Tech Stack section (any dark/abstract video)

## Deploy to Vercel
```bash
npm run build
# Push to GitHub → connect to Vercel → auto-deploys
```
